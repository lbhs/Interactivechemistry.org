 To use this package: 

  1. Go to Edit -> Project Settings...
  2. At the top, fill out the Company Name field as "Interactive Chemistry"
  3. Fill out Product Name field with the name of this product 
  2. Go to the WebGl tab (the icon is a shield with a 5 in it)
  4. Click on the group "Resolution and Presentation"
  5. Choose the IC2020 Responsive template 
  6. In your Assets folder, go to WebGL Templates -> Responsive 
  7. Open the file "index.html" with your preferred text editor 
  8. Replace the needed text with the proper descriptions marked with ALL CAPS
     (locations: at the top [meta description (might show up on google) should be a one liner],
                 in the middle [actual description (will show up on page), should be long],
		 	       optional tag lines and lesson plans can be uncommented for a richer page
